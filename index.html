Thank you for using Canvas Calendar Checker!

This extension enables you to check off tasks that do not have official "completion-criteria" configured by the school course admins. This currently only works on the agenda view within the Calendar tab.

If you appreciate this extension and have the means, feel free to give me a tip, but it's completely optional.

Additionally, you can contribute to the source code on GitHub if it interests you. Feel free to ask for a feature or submit a Pull Request for your own code additions.
